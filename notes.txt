STEPS to Create Framework
----------------------------
1. Install Node.js and VS Code Editor.
2. Create one random root folder.
3. Open folder in VSCode
4. Open terminal in VSCode
5. Run command > npm install playwright@latest
6. Create desired framework folder structure -
        src code folder > config, data, locator, pages, reusable, tests, types
        other folder >  allure-results, excel-report, resources, test-data
        some files > .env file, tsconfig.json, framework-notes.txt
7. Place Utility files in Reusable folder : Actions,ExcelUtility,Helper,Utility,Logger
8. Place Configuration file in Config Folder: GlobalSetup, PlayWrightConfig, TestSetup
9. Extention Used - Allure Support, Playwright Test for VSCode, vscode-icons
10.install Allure Report , .env , Excel XLSX packages
11. install VSCode extentions allure support, Vscode-icon, dotenv, playwright Test for VScode.

Allure Report --
Allure is Java based Reporting Tool
> Install Java on Machine
> Set JAVA_HOME  and Path as %JAVA_HOME%/bin

> Install Allure
npm install -D allure-playwright
npm install -g allure-commandline
npm install -g allure-js-commons

> Install Allure support VS extension
> Configure playwright.config.ts file Reporter -
  reporter:[[ "allure-playwright", { resultsDir: reportPaths.get('allure') }],]

---------------------------Prompt to generate  BasePage Actions------------

Read below function format which contains click action code in playwright - 

 // Method 6 : Click on Element like Button,Link etc.
    async click(element: Locator, desc: string): Promise<void> {
        await test.step(`Click on "${desc}" `, async () => {
            await expect(element).toBeEnabled({ timeout: 10000 });
            await element.click();
        });
    }


Generate at least 30 methods which we can use as actions on different type of web element in same format and consider below pointers
1. Function arguments can be changed based on action requirement.
2. Promise return can be vary based on requirement of function
